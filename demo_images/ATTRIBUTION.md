# Demo Image Attribution

This demo includes two low-volume example images only to make the public schema inspectable before the full benchmark release.

The images are from Fitzpatrick17k, which aggregates public dermatology images from DermaAmin and Atlas Dermatologico under CC BY-NC-SA 3.0. The image files in this folder have been stripped of EXIF metadata and are provided under the same source-image license.

Fitzpatrick17k repository: https://github.com/mattgroh/fitzpatrick17k
